방 자동 삭제 기능 및 유지 시간 설정 기능 추가

주요 변경사항:
- 방 생성 시 유지 시간 선택 기능 추가: 1시간, 3시간(기본), 6시간 중 선택 가능
- 방 목록에 각 방의 남은 시간 표시 (실시간 업데이트)
- 방 내부 접속자 섹션 아래에 방 폭파까지 남은 시간 카운트다운 표시
- 설정한 시간이 지나면 자동으로 방 삭제
- 로비에서 30초마다 방 목록 자동 새로고침으로 남은 시간 실시간 업데이트

기술적 변경:
- server.js:
  * 방 생성 시 expiryHours 파라미터 추가 및 검증 (1, 3, 6시간만 허용)
  * 방 객체에 expiryHours 속성 추가
  * 자동 방 삭제 로직을 저장된 expiryHours 기준으로 변경 (기존 고정 6시간에서 변경)
  * 방 목록 조회 시 expiryHours 포함
  * roomCreated, roomJoined 이벤트에 expiryHours 포함
  * 방 삭제 알림 메시지에 설정된 시간 표시

- dice-game-multiplayer.html:
  * 방 생성 UI에 유지 시간 선택 라디오 버튼 추가 (1시간, 3시간, 6시간)
  * 방 생성 시 선택한 유지 시간을 서버로 전송
  * 방 목록 렌더링 시 저장된 expiryHours 기준으로 남은 시간 계산 및 표시
  * 방 내부 카운트다운 함수에 expiryHours 파라미터 추가
  * initializeGameScreen에서 expiryHours를 받아 카운트다운 시작
  * 로비 섹션 활성화 시 30초마다 방 목록 자동 새로고침 기능 추가
  * roomExpiryInterval, roomListRefreshInterval 변수 추가

UI 개선:
- 방 목록에서 생성 시간 제거, 남은 시간만 표시
- 방 내부 카운트다운이 설정된 유지 시간에 맞춰 정확히 표시
