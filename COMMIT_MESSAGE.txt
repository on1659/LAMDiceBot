주사위 애니메이션 개선 및 게임 시작 검증 추가

주요 변경사항:
- 주사위 애니메이션 개선: 마지막 굴리는 사람에게만 카운트업 애니메이션 재생 (1.5초), 애니메이션 중에는 기록에 저장하지 않고 완료 후 저장
- 종료 메시지 타이밍 조정: 애니메이션 완료 후 1.5초 뒤에 "모든 참여자가 주사위를 굴렸습니다!" 메시지 표시
- 게임 시작 검증: 참여자가 0명이면 게임 시작 불가 및 에러 메시지 표시
- UI 개선: 게임 기록 섹션의 "총 굴림"을 "총 기록"으로 변경

기술적 변경:
- server.js: 게임 시작 시 참여자 수 검증 로직 추가, gameError 이벤트 추가
- dice-game-multiplayer.html: 마지막 굴리는 사람 판단 로직 추가, 애니메이션 시간 1.5초로 조정 (30단계 * 50ms), 애니메이션 완료 후 기록 저장 로직 추가, 종료 메시지 표시 시간 1.5초로 조정, gameError 이벤트 핸들러 추가
