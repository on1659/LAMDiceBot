# Bridge Cross Unused Assets

This folder keeps Bridge Cross files that are not loaded by the current runtime.

- `pixel/`: v1 generated assets replaced by the current stage and sprite sheets.
- `sprites/`: preview images and original loose contact sheets kept for reference.
- `stage/`: target-map v1 materials, target source images, previews, and archived stage variants.
- `debug/`: temporary comparison images created while aligning FX rows.

The active runtime assets remain in `../sprites/` and `../stage/`.
