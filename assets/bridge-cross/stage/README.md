# Bridge Cross Stage Layers

These stage assets are separated so the mockup can draw the scene as composable layers instead of using one baked image.

- `background-void-v2.png`: city void background only, no platforms.
- `start-stage-v3.png`: cropped transparent PNG with the simple start platform only, 728x743.
- `finish-stage-v2.png`: cropped transparent PNG with the finish platform only, 559x794.

`source/` contains chroma-key originals kept for future edits.

Unused v1 target-map layers, target source images, previews, and archived stage variants were moved to `../_unused/stage/`.
