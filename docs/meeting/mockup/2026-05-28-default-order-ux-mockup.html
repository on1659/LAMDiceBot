멘탈 모델 "디폴트는 별도 설정값" "단골 메뉴 = 자주 쓰는 메뉴 중 하나" "디폴트는 옵션, 관리 화면이 따로 있음"
C 방향의 특징
• input 영역 단순화 (A 단점 해소) + 풀 외 메뉴 가능 (B 단점 해소)
• 디폴트 상태는 ⭐의 색/점으로 작게 표시 (A의 항상 표시보다는 약함, 충분히 명확)
• 변경 동작은 모달에서 명시적(A의 멀티버튼 혼란 해소)
• 트레이드오프: 디폴트 변경/해제에 클릭 1회 추가 — 평소 자주 안 바꾸는 동작이라 OK
