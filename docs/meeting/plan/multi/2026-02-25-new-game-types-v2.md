## 1. 1차 회의 리젝 사유

1차 회의에서 나온 폭탄 돌리기, 다수결의 함정, 눈치 게임, 약속의 탑이 전부 리젝됨.

| 리젝 사유 | 설명 |
|-----------|------|
| 조작 횟수 과다 | 여러 라운드, 턴제 → 0~1회 조작이 원칙 |
| 업무 중 부적합 | 전략, 심리전, 실시간 타이밍 → 가볍게 못 함 |
| 순수 운 위반 | 판단/선택이 결과에 영향 → RNG 100%여야 함 |
| 지목 메카닉 | 대인관계 갈등 유발 → 플레이어 간 직접 공격 금지 |

### 절대 원칙 (Golden Rule)
> 0~1회 조작 → 서버가 운으로 결과 결정 → 드라마틱 연출 → 끝

### 7대 체크리스트
| # | 항목 | 기준 |
|---|------|------|
| 1 | 조작 횟수 | 0~1회 |
| 2 | 게임 시간 | 10~25초 |
| 3 | 결과 결정 | 서버 RNG 100% |
| 4 | 3초 룰 | 3초 안에 설명 가능 |
| 5 | 감정 차별화 | 기존 게임과 다른 핵심 감정 |
| 6 | 다시보기 | 결정론적 재생 가능 |
| 7 | 지목 금지 | 플레이어 간 직접 공격 없음 |

## 5. 최종 선정

### 1순위: 지뢰찾기 행운 (Lucky Mine) — 즉시 구현

| 항목 | 내용 |
|------|------|
| 조작 | **1회** (칸 선택) |
| 결과 | 서버 RNG (금/지뢰/안전 위치 사전 배치) |
| 연출 | 13~20초 (선택 8초 + 보드 공개 10~12초) |
| 핵심 감정 | **공간적 서스펜스** — "내가 고른 자리에 뭐가 있을까?" |
| 운의 종류 | **공간의 운** (기존: 숫자/위치/선택의 운과 차별화) |
| 3초 설명 | "칸 하나 고르세요, 금이면 1등 지뢰면 꼴찌" |

**게임 플로우**:
```
1. 방장 시작 → 서버가 그리드 + 금/지뢰 배치 생성
2. 빈 그리드 표시 + "칸을 선택하세요!" (타이머 8초)
3. 각 플레이어 칸 1개 탭 (선택한 칸에 본인 아이콘)
4. 전원 선택 완료 or 타이머 종료
5. 보드 한 줄씩 공개 (금=금빛, 지뢰=코믹 폭발, 안전=체크마크)
6. 결과: 금=1등, 안전=중간, 지뢰=꼴찌
```

**그리드 크기 (플레이어 수 기반)**:
| 인원 | 그리드 | 금 | 지뢰 | 안전 |
|------|--------|-----|------|------|
| 2~3명 | 3x3 | 2 | 2 | 5 |
| 4~5명 | 4x4 | 5 | 5 | 6 |
| 6~8명 | 5x5 | 6 | 6 | 13 |

**7대 체크리스트 통과 확인**:
| # | 항목 | 결과 |
|---|------|------|
| 1 | 조작 0~1회 | PASS (1회 탭) |
| 2 | 10~25초 | PASS (13~20초) |
| 3 | RNG 100% | PASS (서버 사전 배치) |
| 4 | 3초 룰 | PASS |
| 5 | 감정 차별화 | PASS (공간의 운 = 신규) |
| 6 | 다시보기 | PASS (그리드+선택 데이터로 재생) |
| 7 | 지목 금지 | PASS (각자 칸 선택, 대인 공격 없음) |

### 2순위: 슬롯 매칭 (Slot Match) — 추후 검토

| 항목 | 내용 |
|------|------|
| 조작 | **0회** |
| 결과 | 서버 RNG (3릴 심볼 사전 결정) |
| 연출 | 10~12초 (릴 3개 순차 정지) |
| 핵심 감정 | **매칭의 기대감** — "그림이 맞춰질까?" |
| 보류 이유 | 베팅 없이 쾌감 유지 불확실. 지뢰찾기 출시 후 유저 반응 보고 검토 |

## 6. 최종 게임 포트폴리오

| 게임 | 조작 | 연출 | 운의 종류 | 핵심 감정 |
|------|------|------|-----------|-----------|
| 주사위 | 1회 (굴리기) | 3~5초 | 숫자의 운 | 비교의 쾌감 |
| 룰렛 | 0회 | 10~14초 | 위치의 운 | 서스펜스 |
| 경마 | 1회 (말 선택) | 15~25초 | 선택의 운 | 역전 드라마 |
| **지뢰찾기** | **1회 (칸 선택)** | **13~20초** | **공간의 운** | **공간적 서스펜스** |

**4종 모두 Golden Rule 준수: 0~1회 조작 → RNG → 연출 → 끝**
