# 신규 게임 타입 브레인스토밍 회의록

**일시**: 2026-02-25
**참석자 (8명)**: 민준(Creative Director), 서연(UX Planner), 지호(Retention), 하윤(Game Mechanics), 도윤(Trend Analyst), 재현(Backend Dev), 수빈(Frontend Dev), 강현(총괄 PD)
**회의 방식**: 3차 연속 회의 (1차 관전형 → PD 리뷰 → 2차 보완 + 3차 참여형)
**구현 추천 모델**: Opus (다파일 연계, 신규 게임 아키텍처 설계)

---

## 1. 배경

- 인형뽑기 폐기 결정 (경마만큼 재미없음)
- 새 게임 타입 필요 — 재미 + 동기화 + 다시보기 원칙 준수
- 참여형/관전형 포트폴리오 균형 필요

## 2. 인형뽑기 실패 분석

경마 성공 4요인 대비:
| 성공 요인 | 경마 | 인형뽑기 |
|-----------|------|---------|
| 복수 엔티티 동시 경쟁 | O | X |
| 중간 상황 변화 (기믹/날씨) | O | X |
| 클라이맥스 연출 (슬로우모션) | O | X |
| 선택의 무게감 (탈것 선택) | O | X |

## 3. 1차 회의 — 관전형 게임 (기획 5 + 개발 2)

### 검토한 아이디어 (10개)
타워 디펜스, 폭탄 돌리기, 생존 서바이벌, 주식 투자, 보물찾기/지뢰밭, 역경매, 계단 오르기, 배틀로얄 룰렛, 격투 토너먼트, 차트 레이싱

### 투표 결과
| 순위 | 게임 | 점수 | 유형 |
|------|------|------|------|
| 1위 | **폭탄 돌리기** (서바이벌) | 12점 | 관전형 |
| 2위 | **차트 레이싱** (음식 주식) | 9점 | 관전+선택 |
| 보류 | 역경매 | - | 메타 고착 위험 |

## 4. 총괄 PD 리뷰 (강현)

### 핵심 지적 6가지
1. "폭탄 돌리기 = 원형 룰렛 아닌가?" (구조적 동일)
2. 선택의 무게감 부재
3. 소규모(3~4명) 시 라운드 너무 짧음
4. 초반 탈락자 할 일 없음
5. 페이크아웃 반복 피로 (인형뽑기 실패 반복 위험)
6. 포트폴리오 불균형 (참여형 주사위 1개뿐)

### PD 전략 판단
- 경마 개선(Phase 1~4) 먼저 마무리
- 폭탄 돌리기는 보완 후 진행
- 참여형 게임 추가 탐색 필요

## 5. 2차 회의 — PD 지적사항 답변

### 핵심 설계 변경: "도전/패스" 턴제 메카닉

> **변경 전**: 폭탄 랜덤 회전 → 멈추면 탈락 (관전형, 룰렛과 동일)
> **변경 후**: 폭탄 소유자가 매 턴 "도전" 또는 "패스" 선택 (참여형)

| 행동 | 설명 |
|------|------|
| 도전 | 미니게임 수행 → 성공 시 아무 생존자 지목하여 전달 |
| 도전 실패 | 즉시 탈락 (리스크) |
| 패스 | 시계방향 다음 사람, 타이머 -3초 |
| 타이머 만료 | 현재 소유자 탈락 |

### 6대 질문 답변
| # | 질문 | 답변 |
|---|------|------|
| 1 | 참여 메카닉 | 도전/패스 + 지목. 리스크-리워드 의사결정. |
| 2 | 인원 | 4~8명. 세로 리스트형 레이아웃 (원형 포기). |
| 3 | 템포 | 7명 기준 60~80초. 후반 타이머 가속. |
| 4 | 룰렛 차별화 | 매 턴 전략적 행동 + "후회" 감정 발생. |
| 5 | 다시보기 | 하이라이트 타임라인 (라운드별 서머리 + 선택 재생). |
| 6 | 탈락자 | 관전 + 예측 베팅 (다음 탈락자 지목, 적중 시 보상). |

## 6. 3차 회의 — 참여형 게임 탐색

### 포트폴리오 갭
- 부족한 감정: 협동, 추리, 유머, 배신, 공감, 예측
- 부족한 상호작용: 플레이어 간 직접 상호작용

### 검토한 아이디어 (8개)
심판의 눈(마피아), 다수결의 함정(소수결), 눈치 게임, 몸값 경매, 거울 게임(텔레파시), 표정 읽기, 숫자 텔레파시, 약속의 탑(죄수의 딜레마)

### 투표 결과
| 순위 | 게임 | 점수 | 유형 | 핵심 감정 |
|------|------|------|------|-----------|
| 1위 | **다수결의 함정** (소수파 게임) | 19점 | 참여형 | 배신, 예측, 유머 |
| 2위 | **눈치 게임** | 12점 | 참여형 | 긴장, 유머 |
| 3위 | **약속의 탑** (죄수의 딜레마) | 15점 | 참여형 | 배신, 신뢰 |

### 채택 게임 상세

#### 다수결의 함정 (소수파 게임)
- A/B 동시 선택 → 소수파 승리
- 3라운드 x 15초 = 약 45초
- 질문 풀 확장으로 무한 리플레이
- 구현 난이도: 하 (기존 패턴 90% 재활용)

#### 눈치 게임
- 1~N 번호 선점, 동시 선점 시 양쪽 탈락
- 한국 문화 친숙도 100%
- 동시 판정 버퍼(300ms) 새로 구현 필요
- 구현 난이도: 하~중

#### 약속의 탑
- "쌓기(협력)" or "뺏기(배신)" 동시 선택
- 배신 과반 시 탑 붕괴
- 소수결 엔진 100% 재활용, 점수 로직만 변경
- 구현 난이도: 최하 (소수결 이후)
