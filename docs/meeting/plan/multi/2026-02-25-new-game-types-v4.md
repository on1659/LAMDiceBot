## 1. 이전 실패 분석 — 왜 안 끌렸나?

### 3차까지의 문제

태민(PD):
> 3차까지의 문제는 "메카닉에서 출발했다"는 것이다. "폭탄을 돌린다", "카드를 뒤집는다", "계단을 오른다" — 전부 동사가 먼저 나왔다. 끌리는 게임은 **"이런 순간이 있으면 좋겠다"**에서 출발한다.

민준(CD):
> 예능에서 왜 러시안 룰렛이 재미있나? "탕!" 하고 물 맞는 순간. 왜 젠가가 재미있나? 무너지는 순간. **결정적 순간(Climax Moment)이 시각적으로 명확한가?** 이게 핵심이다.

### 실패 패턴 5가지

| 실패 유형 | 예시 | 근본 원인 |
|-----------|------|-----------|
| 숫자 귀결 | 보물뽑기, 운명의카드, 등급가챠 | 결국 "높은 숫자가 이김" = 주사위 변형 |
| 레이스 귀결 | 별쏟아지기, 계단오르기 | 결국 "먼저 도착" = 경마 변형 |
| 위치 귀결 | 서바이벌룰렛, 운세바퀴 | 결국 "어디서 멈추나" = 룰렛 변형 |
| 소거 귀결 | 화산카드, 블랙아웃 | 결국 "순서대로 탈락" = 용암탈출 변형 |
| 가챠 귀결 | 행운의자판기, 타로한장 | 결국 "뭐가 나올까?" = 기대감만, 드라마 없음 |

### 방법론 전환

> **"메카닉 우선" → "감정 우선"**
> "이런 순간이 있으면 좋겠다" → 그 감정을 만드는 메카닉을 역설계

## 2. 감정 역발상 — "소리 지르는 순간" 수집

각 멤버가 2~3개씩 "소리 지르는 순간" 제시:

| 멤버 | 핵심 감정 키워드 |
|------|----------------|
| 승현(카지노) | 올인 쾌감, 시스템이 무너지는 쾌감, 동시 공개 |
| 예진(가챠) | SSR 등급 연출, 지연된 기대감, 아슬아슬함 |
| 준서(스포츠) | 극적 역전, 연쇄 성공의 긴장, 판정 대기 |
| 민준(CD) | 구조적 긴장, 숨기기/들키기, 차등적 공포 |
| 서연(UX) | 언박싱, 점점 다가가는 느낌, 완성 직전 초조함 |
| 하윤(Mechanics) | 선택의 무게+물리적 긴장, 순간적 해소, 터질까말까 임계점 |
| 도윤(Trend) | 보이는것과 다른 진실, 점진적 공개, 파괴와 완성의 경계 |
| 지호(Retention) | 연쇄 성공, 유일한 생존, 공동 운명 |
| 다은(신입) | "아!" 탄식, 아슬아슬한 회피, 매칭의 쾌감 |
| 재현(Backend) | 이진 판정의 쾌감, 장애물 회피, 임계점 서스펜스 |
| 수빈(Frontend) | 붕괴의 스펙터클, "하나만 더!", 도전과 결과의 즉시성 |

### PD 감정 클러스터링

| 클러스터 | 감정 키워드 | 기존 겹침? | 판정 |
|----------|------------|-----------|------|
| **A. 동시 공개** | "뒤집어!" | 새로움 | 탈락 (숫자 귀결) |
| **B. 점진적 공개** | "거의 다 됐어!" | 새로움 | **채택 후보** |
| **C. 연쇄 성공/실패** | "연속!" | 새로움 | **채택 후보** |
| **D. 임계점/붕괴** | "터질까 말까" | 용암탈출과 차별화 필요 | **채택 후보** |
| ~~E. 아슬아슬 회피~~ | "살았다!" | 용암탈출과 유사 | 탈락 |
| ~~F. 역전~~ | "뒤집혔다!" | 경마와 겹침 | 탈락 → 보석으로 부활 |

## 4. 최종 후보 4개 디테일

### 4-1. 다리 건너기 (Bridge Cross)

**한줄**: 유리 다리를 건너서 가장 멀리 간 사람이 이김
**3초 설명**: "유리 다리 건너기. 강화유리 밟으면 성공, 일반유리 밟으면 추락."

| 항목 | 내용 |
|------|------|
| 조작 | 0회 |
| 시간 | 15~22초 |
| RNG | 서버가 각 칸의 강화유리 위치 사전 결정 |
| 핵심 감정 | 연쇄 성공의 긴장 |
| 시각적 킬러 | 유리 깨지며 추락 vs 안전 착지 |
| 끌림 | 오징어게임 유리다리, 한 칸씩 생사 결정 |

### 4-2. 스크래치 대결 (Scratch Duel)

**한줄**: 복권 긁어서 가장 좋은 결과가 나온 사람이 이김
**3초 설명**: "복권 긁기 대결. 3칸 긁어서 매칭 많은 사람이 이김."

| 항목 | 내용 |
|------|------|
| 조작 | 0회 |
| 시간 | 15~20초 |
| RNG | 서버가 복권 심볼 사전 배정 |
| 핵심 감정 | 점진적 기대감 |
| 시각적 킬러 | 복권 긁히는 파티클 + 심볼 매칭 이펙트 |
| 끌림 | 복권 긁기 UX + 타인과 동시 비교 |

### 4-3. 풍선 팽창 (Balloon Inflate)

**한줄**: 가장 크지만 터지지 않은 풍선이 이김
**3초 설명**: "풍선이 불어진다. 크면 이기는데 터지면 꼴찌."

| 항목 | 내용 |
|------|------|
| 조작 | 0회 |
| 시간 | 12~18초 |
| RNG | 서버가 각 풍선의 팽창 곡선 + 터짐 여부 사전 결정 |
| 핵심 감정 | 욕심 vs 공포 |
| 시각적 킬러 | 풍선 팽창 → 떨림 → 균열 → "펑!" 색종이 폭발 |
| 끌림 | "가장 크지만 터진 풍선이 꼴찌" 아이러니 |

### 4-4. 역전의 보석 (Gem Flip)

**한줄**: 보석 3개 순서대로 공개, 매 공개마다 순위가 뒤바뀜
**3초 설명**: "보석 3개 열기. 합산 높으면 이김. 특수 보석이 순위를 뒤집음."

| 항목 | 내용 |
|------|------|
| 조작 | 0회 |
| 시간 | 15~20초 |
| RNG | 서버가 보석 3개 사전 배정 |
| 핵심 감정 | 롤러코스터 순위 변동 |
| 시각적 킬러 | 리더보드 역전 + 특수 보석(왕관/폭탄/2배) 이펙트 |
| 끌림 | 클라이맥스가 3번, 매 단계가 미니 클라이맥스 |

## 7. 최종 게임 포트폴리오 (기존 3 + 신규 5 = 8종)

| 게임 | 조작 | 연출 시간 | 운의 종류 | 핵심 감정 | 감정 곡선 |
|------|------|-----------|-----------|-----------|-----------|
| 주사위 | 1회 | 3~5초 | 숫자의 운 | 비교의 쾌감 | ↑ |
| 룰렛 | 0회 | 10~14초 | 위치의 운 | 서스펜스 | ↑↓ |
| 경마 | 1회 | 15~25초 | 선택의 운 | 역전 드라마 | ↑↓↑↓ |
| **용암 탈출** | **0회** | **15~20초** | **생존의 운** | **상승하는 위기감** | **↑↑↑** |
| **다리 건너기** | **0회** | **15~22초** | **경로의 운** | **연쇄 성공의 긴장** | **↑↑↑↓** |

**감정 곡선 겹치는 것 0개 → 완벽한 포트폴리오 다양성**
