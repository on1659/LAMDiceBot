**일시**: 2026-02-16 (2차)
**주제**: 게임 UX 실질 개선 — 주사위 버튼 부재, 초보자 안내, 채팅 위치/크기
**참석자**: 기획자(A), 개발자(B), QA/UX(C)
**회의 방식**: 3인 멀티에이전트 독립 분석 → 교차 검토 → 합의 도출

## 1. 현황 요약

### 사용자가 직접 지적한 문제
1. **주사위 처음하는 사람이 뭘 해야 할지 모름** — 설명 부재
2. **주사위를 왜 타자로 쳐야 하는지 모름** — 버튼 필요 (타자+버튼 병행)
3. **채팅방이 아래에 있는데 이게 맞는지** — 위치 검토 필요
4. **경마에서 채팅방이 너무 커서 채팅하면서 경마를 볼 수 없음**

### 현재 UX 분석
- **주사위 굴리기**: 채팅에 `/주사위` 입력만 가능. 버튼 없음. 범위 설정 UI 없음.
- **초보자 안내**: 플레이스홀더 1줄만 존재 (`/주사위 - 입력하시면 주사위를 굴릴 수 있습니다.`)
- **채팅 위치**: 게임 섹션 최하단 (스크롤 필요), 주사위 결과는 우측 사이드바에 별도
- **경마 채팅**: max-height 480px, 접기/최소화 없음, 모바일에서 동시 보기 불가

### 사용자 피드백 반영 사항

1. **범위 프리셋 제거**: 원안(1~6, 1~100, 1~1000 프리셋 버튼)은 과함 → **버튼만 추가** (기존 `/주사위` 타자 병행)
2. **경마 채팅 접기 → 라이브 오버레이**: 채팅을 접으면 채팅 자체가 불가능 → **유튜브/아프리카TV 라이브채팅 스타일**로 경마 트랙 왼쪽에 반투명 오버레이 (경마 시야는 가운데~오른쪽이므로 왼쪽 비어있음)
3. **가이드 저장: localStorage → DB 비트플래그**: users 테이블에 `flags INTEGER` 컬럼 추가, 비트 0~3으로 게임별 가이드 완료 추적 (31개까지 확장 가능)
4. **준비/시작 버튼 강조 → 보류**: 시각적 강조보다 배치가 본질적 문제, 레이아웃 개선과 함께 별도 진행

### 의견 충돌 사항

1. **채팅 위치 재배치 vs 현행 유지**
   - 기획자: PC에서 동시 보기 개선 필요 (우측 사이드바)
   - 개발자: 전체 레이아웃 대공사 (16~24시간), 4개 게임 HTML 모두 수정
   - QA/UX: 모바일 슬라이드가 현재 하단 고정보다 나은지 불확실
   - **결론**: 경마 오버레이 먼저 적용 후, 채팅 위치 문제가 여전한지 재평가

2. **퀵 스타트 모드 시기**
   - 기획자: 즉시 (진입장벽 감소 핵심)
   - QA/UX: Phase 1(버튼+가이드+오버레이) 효과 먼저 확인
   - **결론**: Phase 1 채택 항목이 "뭘 해야 할지 모름" 문제를 해결하면 퀵스타트 필요성 재평가
