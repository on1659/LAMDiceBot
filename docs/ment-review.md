# 멘트 검토 (서버 선택·공통 UI)

## 적용한 통일 규칙
- **어미**: 빈 상태/안내 문구는 해요체로 통일 (없어요, 해보세요).
- **띄어쓰기**: "대기중" → "대기 중".

---

## server-select-shared.js (서버 선택)

### 빈 상태·안내
| 위치 | 수정 전 | 수정 후 |
|------|--------|--------|
| 서버 목록 빈 상태 (검색 시) | 검색 결과가 없습니다 | 검색 결과가 없어요 |
| 서버 목록 빈 상태 (기본) | 참여 중인 서버가 없어요<br>검색하거나 새 서버를 만들어보세요! | 참여 중인 서버가 없어요<br>가입한 서버·가입하고 싶은 서버를 검색해보세요. |
| 내 서버 빈 상태 | 만든 서버가 없습니다 | 만든 서버가 없어요 |
| 멤버 목록 빈 상태 | 멤버가 없습니다 | 멤버가 없어요 |

### 뱃지·기타
| 위치 | 수정 전 | 수정 후 |
|------|--------|--------|
| 서버 카드 | 승인 대기중 | 승인 대기 중 |
| 멤버 목록 | 대기중 | 대기 중 |

### 유지한 멘트 (확인용)
- **버튼**: 로그인, 로그아웃, 회원가입, 검색, 취소, 시작!, 입장, 만들기, 닫기, 멤버 관리, 서버 삭제, 승인, 거절, 강퇴
- **placeholder**: 서버 검색..., 이름, 암호코드 (4~6자리 숫자), 닉네임 입력, 참여코드 입력, 2~20자 한글/영문/숫자, 서버 설명 (선택, 100자 이내), 비워두면 공개 서버 (4~20자, 영문/숫자)
- **confirm**: 서버 삭제 시 "정말 삭제하시겠습니까?", 강퇴 시 "님을 강퇴하시겠습니까?" (존댓말 유지)
- **로딩**: 불러오는 중..., 입장 중...
- **에러 모달 제목**: 입장 실패
