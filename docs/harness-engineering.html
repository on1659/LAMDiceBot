하네스 엔지니어링이란?
Claude Code의 훅(Hook) 시스템을 활용해 파일 편집 전후에 자동 검증·피드백을 실행하는 개발 방식. CI/CD처럼 로컬에서도 품질 체크가 자동화된다.
🎯
LAMDice에서의 목적
HTML/CSS 수정 시 CSS 변수 위반·AdSense 누락 등을 Claude에게 즉시 피드백 → 사람이 리뷰하기 전에 자동 자가 수정 가능

하네스 엔지니어링의 핵심 원칙
훅은 Claude가 스스로 실수를 발견하고 수정할 수 있도록 돕는 피드백 루프다. 사람이 리뷰하기 전에 품질 기준을 자동으로 통과하는 것이 목표.
