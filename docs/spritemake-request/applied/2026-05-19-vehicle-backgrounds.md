  - 중단 30% : 중경 (실루엣 또는 단순 디테일)

## 🚨 최우선 원칙: 차량 가독성

**배경은 차량을 돋보이게 하는 무대.** 차량 SVG가 lane 위에 올라가는데, 차량 색과 배경 색이 겹치면 차량이 묻혀서 게임 못 함. 이게 1순위.

각 배경은 해당 차량 색과 **명도(brightness) 또는 색상(hue)** 둘 중 하나로 강한 대비 필수:
- 어두운 차량 (검정 ninja, 갈색/검정 eagle) → 배경은 밝게 또는 밝은 강조 요소 (보름달, 안개, 흰 구름)
- 밝은 차량 (은색 knight) → 배경에 진한 색 요소 (빨간 깃발, 어두운 성탑)
- 같은 hue 차량 (초록 dinosaur, 초록 turtle) → 배경 hue를 보색 또는 황토/주황 톤으로 (화산 용암, 모래)
- 파란 차량 (파란 bird) → 배경 하늘은 노을/석양 톤 (주황-분홍)으로
