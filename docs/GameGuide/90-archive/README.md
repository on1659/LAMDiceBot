# 보관 문서

이 디렉토리의 문서는 **현재 구현 기준이 아닙니다**.

- 줄 번호 기반 설명은 과거 `server.js` 모놀리스 버전 기준입니다.
- 이벤트명, 파일 경로, 라우트가 현재와 다를 수 있습니다.
- 운영 판단에는 `docs/GameGuide/README.md`와 운영 문서를 우선 사용하세요.

## 디렉토리

|------|------|
| `proposals/` | 기능 제안 문서 (미구현) |
